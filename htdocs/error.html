Explanation of Static Repository Gateway Error Codes

  • Error 404 (Not Found): the Static Repositoy is unkown to the Static Repository Gateway
  • Error 502 (Bad Gateway): the Static Repository is not valid
  • Error 503 (Service unavailable): the Static Repository Gateway is busy processing an update of the Static Repository
  • Error 504 (Gateway Timeout): the Static Repository is not accessible by the Static Repository Gateway
